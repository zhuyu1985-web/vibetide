## 2.1 超级个体创作中心（AI辅助创作工作台）

### 产品定位

| 项 | 说明 |
|----|------|
| **功能名称** | 超级个体创作中心 |
| **产品形态** | Web工作台 |
| **使用角色** | 一线编辑、记者、新媒体运营 |
| **上线时间** | Q2 V1 |
| **优先级** | P0 |
| **CCBN展屏** | 第4-6屏 |
| **核心AI员工** | ✍️内容创作师·小文、🎬视频制片人·小剪、🧠频道顾问 |

### 用户痛点

1. 编辑独自面对空白文档，从选题到成稿效率低
2. 图文、短视频、长视频等多种媒介类型需要反复切换工具
3. 不同渠道格式要求不同，适配工作重复且繁琐
4. 历史好用的素材找不到，每次都重新找

### v2.0核心变化：从"创作工具"到"AI团队协作工作台"

旧版本的创作中心是一个编辑器，用户需要手动调用各种AI辅助功能。

新版本中，创作中心是用户与AI员工团队的**协作工作台**：

```
旧版：用户 → [AI写稿按钮] → 得到初稿 → [AI配图按钮] → 得到配图 → ...
新版：用户 → 输入创作目标 → AI团队自动协作 → 用户审核调整 → 完成
```

### 交互流程：目标式创作

```
用户在团队工作台输入目标后，团队工作流到达"内容创作"阶段：

1. 系统自动打开创作中心，预加载：
   - 选题角度（来自💡选题策划师确认结果）
   - 相关素材（来自📦素材管家推荐）
   - 频道风格约束（来自🧠频道顾问）

2. ✍️小文 自动开始创作初稿：
   - 用户可以观看实时生成过程
   - 生成完成后推送消息："初稿完成，请审阅"

3. 用户审阅初稿，可以：
   a. 直接编辑文字（传统编辑器操作）
   b. @小文 用自然语言下达修改指令
   c. 点击段落旁的"AI建议"查看替代方案
   d. 拖入素材管家推荐的素材

4. 如果同时需要视频：
   - 🎬小剪 并行生成视频脚本
   - 用户审核脚本后，小剪开始自动制作
   - 视频制作过程中可插入素材管家的视频片段

5. 内容完成后：
   - 一键提交🔍小审 审核
   - 审核通过后，📡小发 自动适配各渠道格式
   - 用户确认后发布
```

## 2.4 批量生产线

### 产品定位

| 项 | 说明 |
|----|------|
| **功能名称** | 批量生产线 |
| **产品形态** | 任务管理 + 批量处理 |
| **使用角色** | 新媒体运营、内容主管 |
| **上线时间** | Q3 V1 |
| **优先级** | P1 |

| **核心AI员工** | ✍️内容创作师 + 📡渠道运营师 |

### v2.0核心变化

批量生产从"用户配置规则的流水线"变成"用户告诉AI团队一个批量目标"：

```
旧版：配置模板 → 导入数据 → 逐条生成 → 逐条审核
新版：用户告知目标"每日早间新闻9条，基于昨日6条已审核稿件适配9个渠道"
     → AI团队自动完成
```

## 2.5 节赛会展自动产线（原2.6）

### 产品定位

| 项 | 说明 |
|----|------|
| **功能名称** | 节赛会展自动产线 |
| **产品形态** | 事件驱动自动化流水线 |
| **使用角色** | 体育编辑、会议报道组、大型活动报道团队 |
| **上线时间** | Q3 V1 |
| **优先级** | P1 |
| **CCBN展屏** | 第11屏 |
| **核心AI员工** | 🎬视频制片人·小剪（赛事模式） |

### 场景说明

节日、赛事、会议、展会等场景的特点：**大量素材短时间涌入，需要快速加工分发。**

例如：体育赛事直播中，每个进球都需要在3分钟内生成短视频集锦分发到多渠道。

### v2.0核心变化

赛事场景下，🎬视频制片人·小剪 切换到"赛事模式"，自主能力大幅提升：

```
赛事模式下的小剪:
- 权限等级从executor升级为coordinator
- 自动监听直播流，检测精彩时刻
- 自动拆条、自动配解说字幕、自动生成集锦
- 自动分发到各渠道（用户预先授权）
- 全程自动执行，用户只需在赛后审核
```

### 功能需求清单

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| F2.5.01 | 事件预设 | 预设事件信息（赛程/会议议程/活动日程） | P1 |
| F2.5.02 | 直播流接入 | 接入直播流作为实时素材源 | P1 |
| F2.5.03 | 精彩时刻检测 | 🎬小剪 实时检测进球/红牌/重要发言等精彩时刻 | P1 |
| F2.5.04 | 自动拆条 | 精彩时刻自动拆条，生成短视频 | P1 |
| F2.5.05 | 自动解说字幕 | 自动为拆条内容添加解说字幕 | P1 |
| F2.5.06 | 自动集锦 | 赛事/会议结束后自动生成精彩集锦 | P1 |
| F2.5.07 | 快速分发 | 生成后自动分发到预设渠道 | P1 |
| F2.5.08 | 事件数据看板 | 实时展示事件相关内容的产出和传播数据 | P1 |

### 赛事模式工作流

```
赛前准备（用户设定）:
├── 输入赛事信息：中超 · 某队 vs 某队 · 19:30开球
├── 启动小剪赛事模式
├── 预设分发渠道和格式要求
└── 设定自动执行权限（无需逐条审核）

赛中自动执行:
├── 🎬小剪 监听直播流
├── 检测到进球 →
│   ├── 自动拆条（前15秒+后15秒）
│   ├── 自动配字幕："XX队 XX球员 第35分钟进球"
│   ├── 自动适配格式（竖版抖音 + 横版微信视频号）
│   └── 📡小发 自动发布（3分钟内完成全链路）
├── 检测到红牌 → 同上流程
├── 半场结束 → 自动生成上半场集锦
└── 全场结束 → 自动生成全场精彩集锦

赛后总结:
├── 📊小数 推送赛事内容传播效果报告
├── 🎬小剪 生成赛事内容汇总
└── 用户审核赛后内容，确认归档
```
