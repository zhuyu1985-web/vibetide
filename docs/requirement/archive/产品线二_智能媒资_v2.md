## 1.1 媒资智能理解中心（Asset Intelligence Center）

### 产品定位

| 项 | 说明 |
|----|------|
| **功能名称** | 媒资智能理解中心 |
| **产品形态** | 后台处理引擎 + 管理看板 + 📦素材管家的底层能力 |
| **使用角色** | 媒资管理员（管理看板）、一线编辑（通过📦小资间接使用） |
| **上线时间** | Q1 |
| **优先级** | P0 |
| **CCBN展屏** | 屏1 |
| **支撑AI员工** | 📦素材管家·小资 — 本功能是小资"看懂素材"的核心能力 |

### 用户痛点

- 媒资库数十万至百万级资源，"存得多、用不动"
- 编辑想找一段特定画面（如"2019年长江大桥的航拍"），需人工翻找数小时
- 视频内的关键场景、人物、对话没有结构化标注
- **核心问题：媒资 = 仓库，而非媒资 = 金矿**

### v2.0核心变化

旧版本中，用户需要到"媒资管理后台"手动搜索和浏览AI理解结果。

新版本中，编辑**不再直接操作媒资理解后台**，而是在任何工作场景中与📦素材管家·小资对话：

```
旧版：编辑 → 打开媒资管理 → 输入关键词搜索 → 浏览结果 → 手动选择
新版：编辑 → @小资 "找一段去年两会养老金报道的采访画面"
      📦小资 → "找到了！这3条匹配度最高，要推给✍️小文吗？"
```

媒资理解中心的核心价值从"给编辑用的搜索工具"转变为**"让📦小资真正看懂每条素材的底层引擎"**。

### v2.0核心变化：从"配置Agent"到"培养一个懂频道的AI顾问"

```

┌──────────────────┐              ┌──────────────────────────────┐
│ Channel Agent     │              │  🧠 频道顾问·新闻老陈         │
│ Studio           │              │                              │
│ · 配置System     │              │  "我是新闻频道的AI顾问。      │

### 核心概念：频道顾问是"被培养出来的"

创建一个🧠频道顾问的过程类似"入职培训"：

```
第1步：了解频道（自动分析历史内容 → 生成内容基因画像）
       ↓ "看看这个频道过去做了什么，做得好的是什么"
       
第2步：学习知识（灌入垂直领域知识库 → 掌握专业知识）
       ↓ "学习这个领域的专业知识、政策法规、行业术语"
       
第3步：塑造性格（配置沟通风格、用词偏好、禁忌事项）
       ↓ "新闻频道的顾问要严肃理性，娱乐频道的顾问要活泼有趣"
       
第4步：试岗验证（测试对话，验证顾问输出质量）
       ↓ "给同一个热点，看不同频道的顾问建议是否有差异"
       
第5步：上岗工作（绑定到团队，在创作流程中提供风格把关）
```

## 1.3 资产盘活引擎（Asset Revive Engine）

### 产品定位

| 项 | 说明 |
|----|------|

| **使用角色** | 一线编辑（通过📦小资获得推荐） |

| **CCBN展屏** | 屏3 |

### v2.0核心变化

```

      · 每日晨会前推送"今日值得翻出来的旧宝贝"
      · 团队接到新选题时自动匹配历史素材
      · 发现热点与历史内容高度关联时主动提醒
      · 编辑@小资 主动查询时推荐
