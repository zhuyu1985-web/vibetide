## 1 文档概述

### 1.1 文档目的

本文档是 VibeTide 潮汐·数智全媒平台中「智能媒资」模块的产品需求文档，旨在基于现有操作手册和产品能力，系统性梳理智能媒资的功能范围、交互流程和业务规则，为后续的产品迭代和开发提供明确的需求基线。

### 1.2 适用范围

- **产品线：** 数智全媒产品平台 —— 智能媒资管理模块
- **目标客户：** 广电媒体、融媒体中心、高校媒体、政企媒体机构
- **读者：** 产品经理、前后端开发、测试、项目交付、售前

### 1.3 名词定义

| 术语 | 定义 |
|------|------|
| 媒资 | 媒体资产，包括视频、音频、图片、文档、文稿等各类数字内容资源 |
| 编目 | 对媒资进行结构化标注，包括标题、关键词、描述等元数据 |
| 成品库 | 组织级资源管理空间，支持树状栏目、权限控制 |
| CDN分发 | 将视频资源发布到内容分发网络，实现加速播放 |
| CMS推送 | 将资源推送到内容管理系统进行发布 |
| 转码组 | 预设的视频/音频转码参数组合，包括码率、分辨率等 |
| 华生 AI | 平台自有 AI 模型底层，提供标签/人脸/语音/字幕/内容审核能力 |
| 意图检索 | 基于语义理解的智能搜索，支持自然语言和图片搜索 |

## 2 产品概述

### 2.1 产品背景

融媒体作为十四五规划的国策，在数字中国政策及各行业"AI+"打造新质生产力的趋势下，智能融媒体、智能全媒体成为新的发展方向。平台基于自有华生 AI 模型底层及 AI Agent 智能体模式，重新定义媒体生产及传播模型的"汇聚、沉淀、生产、审核、传播、分析"模块能力。

### 2.2 产品定位

智能媒资是数智全媒产品平台的核心模块，承担媒体资产的全生命周期管理，包括资源汇聚、存储、AI智能分析、编目、检索、审核、分发、共享等全流程能力。主要服务于广电媒体、融媒体中心、高校和政企媒体机构。

### 2.3 核心价值

1. **多层级资源库架构：** 个人库、公共库、成品库、版权库、知识库、共享库多维度组织资源
2. **AI Native 智能处理：** 基于华生 AI 实现标签/人脸/语音/字幕/内容审核自动化
3. **语义级检索：** 意图检索 + 以图搜图 + 全文检索，大幅提升素材查找效率
4. **全流程资产管理：** 从汇聚入库→AI分析→编目→审核→分发的完整链路

## 4 详细需求规格

### 4.1 资源库架构

资源库采用多库分区架构，左侧栏目树导航，右侧资源列表展示，各库定位如下：

| 资源库 | 定位与职责 | 关键规则 |
|--------|-----------|----------|
| 全部资源 | 首页门户，提供搜索入口、推荐、宣传日历、专题 | 仅展示，不管理资源 |
| 个人库 | 用户私有素材空间 | 可设置公开/不公开，公开资源自动进入公共库 |
| 公共库 | 汇聚所有用户公开资源 | 只读，支持下载和复制链接 |
| 版权库 | 版权资源统一管理 | 展示权利项、到期提醒（30天）、下载需确认版权限制 |
| 成品库 | 组织级资源管理 | 支持多级栏目、权限、工作流、转码组绑定 |
| 知识库 | 按主题汇聚资源 | 公共知识库(10类) + 自定义知识库 |
| 共享库 | 定向共享资源 | 指定用户/角色/组织/群组，支持到期时间 |
| 回收站 | 已删除资源暂存 | 支持还原/彻底删除/清空 |

### 4.2 成品库栏目管理

成品库是智能媒资的核心管理空间，通过多级栏目树组织资源。

#### 4.2.1 栏目创建

- 支持在成品库根节点创建一级栏目，在任意栏目下创建子栏目
- 创建时仅需输入栏目名称，点击确定即完成
- 栏目层级不限，但建议不超过 5 级

#### 4.2.2 栏目设置

- **绑定工作流：** 资源提审时默认走此工作流
- **指定视频转码组：** 子栏目继承，未指定则用系统默认
- **指定音频转码组：** 同上
- **栏目权限：** 按角色设置栏目及资源的访问权限
- **栏目绑定编目模板：** 分栏目模板和素材模板，子栏目继承父栏目

#### 4.2.3 栏目其他操作

- **重命名：** 悬浮出现操作按钮，弹窗修改
- **排序：** 手动调整同级栏目顺序
- **收藏：** 加入栏目收藏夹，快速定位
- **专辑编目：** 设置栏目标题和封面图
- **删除：** 仅支持删除空栏目（无子栏目且无资源），不可恢复
- **搜索：** 输入关键词快速定位栏目

### 4.3 资源检索筛选

#### 4.3.1 意图检索（AI 语义搜索）

通过对用户输入内容进行语义级理解，准确识别用户核心需求与目标。

- 支持自然语言输入，AI 解析意图后进行向量数据库匹配
- 支持上传图片进行以图搜图，匹配标题、标签、语音、字幕等多维度信息

#### 4.3.2 普通检索

- **全部检索：** 跨基础编目/标签/人脸/语音/字幕/作者/自定义编目统一搜索
- **标题检索：** 仅匹配资源标题
- **标签检索：** 仅匹配资源标签

#### 4.3.3 筛选与排序

- **资源类型筛选：** 全部/视频/音频/图片/文档/文稿
- **排序：** 创建时间、文件名称、文件大小，各支持正序/逆序
- **高级检索（漏斗图标）：** 创建人、上传时间、来源类型、AI分析状态、AI审核状态、CDN分发状态、CMS推送状态、审核状态、编目状态、转码状态、时长

### 4.4 资源列表展示

#### 4.4.1 卡片模式（默认）

展示内容：

- **封面图：** 视频资源支持鼠标左右移动快速预览视频帧
- **状态标识：** 版本号、时长、AI状态、审核状态、CDN状态、CMS状态、编目状态
- **资源名称、创建人、创建时间**
- **悬浮操作按钮：** 复制标题、更多操作

#### 4.4.2 列表模式

- 表格样式展示，可自定义展示字段及顺序
- 展示字段：封面图、资源名称、创建人、创建时间、资源状态、栏目、操作

#### 4.4.3 文件信息面板

- 点击右上角图标展开右侧面板，展示选中资源的基本信息、转码信息
- 视频资源可在面板中快速设置封面图（上传本地图片或视频截图）
- 可查看不同清晰度的文件信息，复制播放地址，下载文件

#### 4.4.4 分页

- 默认每页 20 条，可设置 10/50/100 条
- 支持页码点击和跳转

### 4.5 资源操作

#### 4.5.1 上传

支持三种上传方式：

- **文件上传：** 选择本地文件，支持视频/音频/图片/文档/文稿多种格式
- **文件夹上传：** 整个文件夹批量上传
- **百度网盘上传：** 扫码绑定后，选择网盘资源同步入库

上传规则：

- 可选择视频转码组、音频转码组、密级设置
- 支持上传前重命名和删除文件
- 支持去重提示（检测已存在的文件）
- 实时展示上传进度和转码进度

#### 4.5.2 分享

- 支持单个和批量分享
- 分享设置：允许下载、密码保护（4位数字）、过期时间、限制次数
- 链接分享 + 二维码分享
- 分享管理：查看/编辑/取消/删除分享记录

#### 4.5.3 智能处理

资源入库后自动发起 AI 分析，也支持手动重新发起（消耗 AI 资源费用）。按资源类型支持不同能力：

| 能力 | 视频 | 音频 | 图片 | 文稿 |
|------|------|------|------|------|
| 标签信息 | ✓ | ✓ | ✓ | ✓ |
| 人脸信息 | ✓ | — | ✓ | — |
| 语音信息 | ✓ | ✓ | — | — |
| 字幕信息 | ✓ | ✓ | ✓ | — |
| 机审确认 | ✓ | ✓ | ✓ | ✓ |
| 机审疑似 | ✓ | ✓ | ✓ | ✓ |

#### 4.5.4 批量操作

选中多个资源后顶部弹出批量操作栏，个人库与成品库支持的批量操作不同：

- **个人库：** 全选/取消全选/删除/分享/入成品库/智能处理
- **成品库：** 全选/取消全选/删除/分享/移动/CDN分发/推送CMS/提交审核/智能处理/编目

#### 4.5.5 其他资源操作

| 公开/不公开 | 公开到公共库 / 取消公开 | 个人库 |
| 移动 | 跨栏目移动资源，支持批量 | 成品库 |

| CDN分发 | 视频分发到CDN厂商，需定制 | 成品库 |
| 推送CMS | 视频推送到CMS厂商，需定制 | 成品库 |
| 关联版本 | 上传新版本绑定到现有资源，版本号递增 | 成品库 |
| 补码 | 对已入库视频/音频追加转码码率 | 成品库 |
| 工具生产 | 挂接轻量化生产工具，按角色权限展示 | 通用 |
| 复制文本链接 | 复制多码率播放/下载地址 | 通用 |

### 4.6 资源详情页

点击资源进入详情页，左侧为播放器/预览区，右侧为多 Tab 信息展示区。

#### 4.6.1 编目

- 入库后默认"未编目"状态
- 填写编目字段并保存后状态变为"已编目"
- 编目字段受栏目绑定的编目模板控制

#### 4.6.2 AI 信息

- **标签信息：** 展示 AI 识别标签，支持删除和新增标签（选择分类+输入名称）
- **人脸信息：** 展示识别人物及置信度，点击人物查看出现片段，点击片段跳转视频位置
- **语音信息：** 语音转文字，支持编辑、查找/替换，点击内容跳转对应时间点
- **字幕信息：** OCR 字幕识别，点击跳转对应位置
- **机审确认/疑似：** 内容安全检测结果，点击跳转查看

#### 4.6.3 其他详情功能

- **标记点：** 视频/音频打标记点，支持增删改，点击跳转
- **媒体属性：** 展示原格式及各码率的详细信息
- **基本信息：** 栏目/类型/上传人/时间/来源/审核/编目/密级
- **意见批注：** 视频画面标注批注，支持回复、复制、删除
- **审核意见：** 工作流审核流程及意见展示
- **版权信息：** 展示关联的详细版权权利项
- **自定义封面：** 上传封面图、视频截图、海报图（16:9/9:16/1:1）

### 4.7 版权库

- 展示系统全部版权资源，包含版权权利项、使用时间、地域、范围、渠道信息
- 版权到期提醒：30 天内到期和已到期在页面标识提示
- 下载时弹出版权使用限制提示，用户确认后方可下载
- 支持查看版权资源的下载记录
- 支持按版权名称、类型、所属栏目、状态检索

### 4.8 知识库

#### 4.8.1 公共知识库

将资源和知识进行关联，预置 10 类知识库：人物库、事件库、节日节气、场景汇聚、重要讲话、敏感库、纪念日、专家库、专家讲座、机构库。

#### 4.8.2 我的知识库

- 用户自建知识库，设置筛选条件自动汇聚资源
- 条件项：名称、文件大小、标签、创建日期、类型、宽度、高度、时长等
- 汇聚范围为整个素材库中有权限的素材

### 4.9 共享库

- 支持对资源指定共享范围：公开 / 指定用户 / 指定角色 / 指定组织 / 指定群组
- 可设置共享到期时间，到期后不影响已取用资源
- **共享给我：** 查看/取用/下载（取用即复制一份到目标栏目）
- **我的共享：** 管理已共享资源，编辑范围/取消共享/查看取用记录

## 5 非功能需求

### 5.1 性能需求

- 资源列表加载时间 < 2 秒（万级资源规模）
- 意图检索响应时间 < 3 秒
- 文件上传支持断点续传，单文件不限大小
- AI 智能处理异步执行，不阻塞用户操作

### 5.2 兼容性

- **浏览器：** Chrome 90+、Edge 90+、Firefox 90+、Safari 15+
- **视频格式：** mp4, avi, ts, mkv, mov, wmv, flv, mpg, mxf, rmvb, 3gp, webm, m4v, mpeg 等
- **音频格式：** mp3, aac, wav, flac, ogg, wma, m4a 等
- **图片格式：** jpg, png, gif, bmp, webp, tiff, svg, raw 等
- **文档格式：** doc, docx, pdf, ppt, pptx, xls, xlsx 等

### 5.3 安全需求

- 资源访问基于角色权限控制（RBAC）
- 资源密级管理：公开/秘密/不公开/绝密/机密
- 分享链接支持密码保护、过期时间、访问次数限制
- 版权资源下载需确认版权限制条款

## 6 数据状态定义

资源在系统中存在多维度状态，以下为各状态的定义：

| 状态维度 | 可能值 | 说明 |
|----------|--------|------|
| AI分析状态 | 未发起/分析中/分析成功/分析失败 | 入库后自动发起 |
| AI审核状态 | 未发起/审核中/审核成功/审核失败 | 内容安全检测 |
| 审核状态 | 待审核/审核中/审核成功/审核打回 | 人工工作流审核 |
| CDN分发状态 | 未开始/进行中/分发成功/分发失败/已撤销 | 视频CDN发布 |
| CMS推送状态 | 未开始/进行中/推送成功/推送失败/已撤销 | CMS内容推送 |
| 编目状态 | 未编目/已编目 | 填写并保存编目信息后变更 |
| 转码状态 | 未开始/进行中/转码成功/转码失败/已取消 | 上传后自动转码 |
| 密级 | 公开/秘密/不公开/绝密/机密 | 上传时设置 |
