# 媒体字典账号矩阵 — 录入操作手册

> 给数据录入同事的工作说明。当前共 **113 个媒体**,需要补全各平台账号。
>
> 配套文件:
> - `media-outlets-channels-todo.csv` — 待填表(可用 Excel/WPS 打开)
> - 本文档 — 操作说明 + 各平台 ID 提取规则

---

## 你要做什么

打开 `media-outlets-channels-todo.csv`,每行是一个媒体。你需要给**有公开账号的平台**填上**主页 URL**(不是 ID,是 URL):

| 列 | 你填什么 | 不需要时填什么 |
|---|---|---|
| 抖音主页URL | `https://www.douyin.com/user/xxx` | 留空 |
| 微博主页URL | `https://weibo.com/u/xxx` 或 `https://weibo.com/xxx` | 留空 |
| 快手主页URL | `https://www.kuaishou.com/profile/xxx` | 留空 |
| 公众号ghid | `gh_xxx`(只有这列需要填**ID**) | 留空 |
| 备注 | 异常情况说明(如"找不到官方号""有多个矩阵号"等) | 留空 |

**不要填 ID 字段**(sec_uid / uid / userId 这些)——系统会从你填的 URL 自动解析。

---

## 各平台找账号的方法

### 抖音 → 抖音主页URL

1. 浏览器打开 `https://www.douyin.com`(需登录,可用手机扫码)
2. 搜索栏输入媒体名(如"人民日报")
3. 点 **用户** Tab → 找带 **官方认证** ✓ 蓝标的账号(注意是蓝标"政务"或"媒体"认证,不是 V 标)
4. 点进主页 → **复制地址栏完整 URL** → 粘到 CSV

**鉴别要点**:
- 中央级媒体(人民日报/央视等)的官方号一定有"政务认证"蓝标
- 同一媒体可能有多个矩阵号(如"人民日报"+"人民日报评论"),都填上,用 ` | ` 分隔
- 找不到带认证的号 → 留空 + 备注"无官方认证抖音号"

### 微博 → 微博主页URL

1. 浏览器打开 `https://weibo.com`(需登录)
2. 搜索媒体名
3. 找带**橙 V**(媒体官方)或**蓝 V**(政务)认证的账号
4. 点进主页 → 复制 URL(形如 `weibo.com/u/2803301701` 或 `weibo.com/rmrb`)

**鉴别要点**:
- 微博认证比抖音规范,基本能找到对应官方号
- 复制完整 URL,不要只复制 uid 数字

### 快手 → 快手主页URL

1. 浏览器打开 `https://www.kuaishou.com`
2. 搜索媒体名
3. 不是所有媒体都开快手号,**找不到很正常,留空 + 备注**
4. 找到 → 点进主页 → 复制 URL(形如 `kuaishou.com/profile/3xy4nh...`)

**预期覆盖率**:大概只有 30-40% 媒体有快手号,主要集中在央视/省级电视台/部分区县融媒。

### 公众号 → ghid(这是唯一要填 ID 的列)

公众号没有标准 URL 主页,只能填 `gh_xxx` 格式的 ID。三种获取方法:

**方法 A:微信 PC 端 → 文章详情**
1. 微信 PC 客户端打开任意一篇该公众号的文章
2. 右上角"..." → 复制文章链接
3. 粘到浏览器打开 → 右键查看源码 → Ctrl+F 搜 `gh_` → 找到 `__biz=` 后那段 base64 解码也能拿,但更简单的是看 `gh_xxxxx`

**方法 B:第三方工具(推荐)**
- 西瓜助手(微信公众号查询工具)
- weixin.sogou.com 搜公众号 → 进入 → 二维码图片 URL 里含 ghid
- 老牌工具 newrank.cn 也有公开 ghid 数据

**方法 C:直接放弃**
公众号 ghid 不好找。**找不到就留空**,系统会把这条 channel 作为"反查标识"(知道这个公众号存在,但无法自动采集),实际工作不影响。

## 录完之后

把填好的 CSV 发回研发,我会:
1. 跑脚本把 CSV 解析回数据库的 `channels` jsonb 字段
2. URL 字段用 `parseAnyChannelUrl()` 自动提取 secUid/uid/userId
3. 公众号 ghid 直接写入 `channels[type=wechat_oa].ghid`
4. 入库后你可以登 vibetide 媒体字典页验收

---

## 优先级建议

如果时间有限,**按这个顺序填**:

1. **央级 12 家**(人民日报/新华社/央视等) — 必填,因为研究场景重点
2. **省/市级 8 家**(重庆媒体) — 重点
3. **区县融媒 70+ 家** — 抖音/微博比较常见,公众号几乎都有
4. **行业媒体 12 家** — 公众号优先,社媒覆盖率低
5. **政务新媒体** — 主要是公众号
