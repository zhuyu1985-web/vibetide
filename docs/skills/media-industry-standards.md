## 2. 新华社发稿规范

新华社 2024 版《发稿规范》核心要求（中央媒体内容生产参照）：

### 2.1 政治表述

- 党和国家领导人姓名职务必须**严格对照官方最新称谓**，不得简写、错序、别称
- 重要会议首次出现用**全称**（如「中国共产党第二十届中央委员会第三次全体会议」），简称限于同文重复出现
- 政策原文引用必须**完整保留原意**，不得断章取义
- 涉外用语遵循《中国特色话语对外翻译标准化术语库》
- 港澳台相关表述严格对照《中央政府涉港澳台报道用语规范》

### 2.2 数字使用

- 阿拉伯数字用于：时间、数量、编号、序数词
- 汉字数字用于：成语/习语、概数、时代名、朝代、岁数
- 涉及金钱：「8.2 亿元」而非「820000000 元」；官方数据保留原单位

### 2.3 标点

- 书名号限于书/报/刊/文件/法律/章节，不得用于普通名词
- 引号直接引用原话，间接引语不加引号
- 破折号表示解释说明/语义转折，不得滥用

### 2.4 禁止用词（2024 年更新）

- 避免：「本报讯」「特讯」（除非确有独家采访）
- 避免：「我国」改「中国」（国际新闻除外）
- 避免：「首次」「第一次」「唯一」等绝对化（除非有官方权威背书）
- 避免：「开会」（改用「召开会议」）

## 3. 中央媒体用语白名单/黑名单

### 3.1 白名单（推荐使用）

**动词**：表示 / 指出 / 强调 / 部署 / 提出 / 要求 / 宣布 / 签署 / 发布 / 考察 / 调研 / 会见 / 审议

**名词**：重要指示 / 重要批示 / 讲话 / 报告 / 声明 / 协议 / 决定 / 意见 / 方案 / 部署

**形容词**：深入 / 全面 / 系统 / 深刻 / 明确 / 切实 / 扎实 / 有力

### 3.2 黑名单（禁用）

**情绪化**：震惊 / 爆 / 惊天 / 重磅（作为标题炒作）
**炒作化**：颠覆 / 吊打 / 碾压 / 秒杀 / 血洗
**极限化**：最好 / 第一 / 顶级 / 唯一 / 绝对 / 100%（广告法第 9 条）
**贬损化**：砸盘 / 跳水 / 崩盘（除特定财经专业语境）
**娱乐化**：吃瓜 / 翻车 / 塌房（时政稿严禁）

## 4. 广告法极限词清单（禁用词）

《广告法》第九条明确禁止的极限用语（2021 修订）：

### 4.1 时间极限

国家级 / 世界级 / 全球级 / 宇宙级 / 最强 / 巅峰 / 极限 / 绝无仅有

### 4.2 程度极限

最 / 第一 / 独家 / 独一无二 / 首个 / 唯一 / NO.1 / TOP1 / 王牌 / 领袖

### 4.3 权威极限

国家级领导人推荐 / 特供 / 专供 / 专家推荐（除非有正式合规背书）

### 4.4 频度极限

绝对 / 100% / 纯天然 / 零添加（食品类除非有权威检测报告）

### 4.5 保证类

保证 / 保障 / 承诺 / 稳赚 / 无风险 / 100% 投资回报

### 4.6 违反罚则

《广告法》第 57 条：发布禁用极限词广告，由市场监督管理部门责令停止发布，
对广告主处以 **20 万-100 万元罚款**；情节严重的吊销营业执照、吊销广告发布登记证件。

## 5. 时政稿件敏感红线

中宣部、网信办 2024 年时政稿件内容规范（内部通告整理）：

### 5.1 绝对禁止

- 党和国家领导人姓名职务错序或不规范
- 擅自使用领导人**未公开讲话内容**或**未发布的政策草案**
- 涉及中央政治局常委个人的非官方报道
- 涉及两岸四地的非中央定调表述

### 5.2 严格限制

- 涉军信息必须有官方发布源
- 涉及重大事故必须有政府通报后才能报道
- 涉及境外反华势力引述必须批判性表述
- 涉及民族宗教必须对照《民族宗教政策法规手册》

### 5.3 处理流程

时政稿件发布前必须经**三审三校**：
1. **初审**：记者/编辑自审（对照负面词清单）
2. **二审**：部门主任/编辑部主任
3. **三审**：总编辑/值班总编（重大政治稿件）

自动化工作流中 `quality_review` 的 `strict` 档位必须通过 95+ 分
+ 全部合规红线扫描通过才能入 `/articles` 表。

## 6. 未成年人保护条款

《未成年人保护法》第 49 条、第 51 条关键要求：

### 6.1 信息披露限制

- 不得披露未成年人**真实姓名**（特别是涉罪 / 受害 / 医疗相关）
- 不得披露未成年人**真实身份信息**（学校、班级、住址）
- 不得披露未成年人**脸部清晰图像**（相关报道必须模糊处理）
- 不得利用未成年人形象做商业代言（除非监护人书面同意 + 监管审查）

### 6.2 内容禁区

- 不得以任何形式向未成年人推广烟、酒、赌博、暴力、色情
- 不得制作/传播含有未成年人不雅、危险行为的内容
- 不得报道未成年人自杀、自残等行为的细节

### 6.3 自动化合规

`fact_check` / `quality_review` 检测到涉未成年人报道时，必须：
- 检查是否有真实姓名/身份信息 → 命中立即拒发
- 检查是否有清晰面部 → 命中强制打码提示
- 检查是否报道细节（自杀/自残方法） → 命中立即拒发

## 7. 网络信息内容生态治理规定

网信办 2020 年《规定》定义的三类内容：

### 7.1 鼓励内容（第 6 条）

- 习近平新时代中国特色社会主义思想
- 党和国家大政方针
- 社会主义核心价值观
- 中华优秀传统文化
- 科学技术进步

### 7.2 禁止内容（第 7 条 · 违法信息）

- 反对宪法确定的基本原则
- 危害国家安全、泄露国家秘密、颠覆国家政权、破坏国家统一
- 损害国家荣誉和利益
- 煽动民族仇恨、民族歧视
- 破坏国家宗教政策、宣扬邪教和封建迷信
- 散布谣言、扰乱经济秩序和社会秩序
- 散布淫秽、色情、赌博、暴力、凶杀、恐怖或教唆犯罪
- 侮辱诽谤他人、侵害他人合法权益
- 含有法律、行政法规禁止的其他内容

### 7.3 不良内容（第 8 条 · 有害信息）

- 炒作绯闻、丑闻、劣迹
- 渲染血腥暴力
- 宣扬低俗庸俗媚俗
- 过度娱乐化
- 标题党
- 侵害未成年人合法权益

**网信办日常巡查命中不良内容的后果**：平台责令整改；持续违规可下架账号；
严重违规移送司法。

## 8. 信源权威度分级

内容生产中信源引用的权威度分级（自高到低）：

### S 级（最高权威）

- 新华社
- 人民日报
- 中央广播电视总台（CCTV / CGTN / CNR）
- 国务院新闻办
- 国务院各部委官网 / 官方客户端
- 各省级人民政府官网

### A 级（一级媒体）

- 光明日报 / 经济日报 / 中国日报
- 各省级党报党媒
- 中国新闻社
- 中国网 / 求是网 / 学习强国

### B 级（主流商业媒体）

- 21 世纪经济报道
- 第一财经
- 澎湃新闻
- 界面新闻
- 财新网 / 财经杂志
- 36Kr / 钛媒体

### C 级（社交平台自媒体）

- 微博头部账号（需判断是否机构认证）
- 微信公众号（需判断是否机构认证 / 企业认证）
- 抖音头部账号
- 小红书、B 站
- 头条号

### D 级（待验证）

- 未实名的匿名信源
- 未认证的自媒体
- 境外 Twitter / X 转发内容
- 匿名"知情人"爆料

### 引用规范

- **时政稿件**：仅可引用 S 级 + A 级
- **深度稿件**：S-B 级，C 级需注明「据 XX 报道」
- **娱乐轻资讯**：S-C 级
- **D 级**：任何稿件禁止作为事实来源；仅可作为「存在某某传闻」的陈述对象

## 9. AI 生成内容合规要求

《生成式人工智能服务管理暂行办法》（网信办、国家发改委等七部门 2023 发布）：

### 9.1 标注义务

使用 AI 生成的新闻内容**必须显著标注**为 AI 辅助生成（《办法》第 12 条）。
VibeTide 当前实现：
- 文章末尾 footer 自动添加「本文由 VibeTide AI 辅助生成，已经人工审核」
- CMS 入库时 `articles.meta.aiGenerated = true`

### 9.2 事实核查义务

AI 生成的新闻报道必须经过**人工复核**，特别是：
- 具体数字 / 人名 / 时间 / 地点
- 政策原文引用
- 专家观点引用
- 医疗、金融、法律等专业领域

VibeTide `fact_check` skill 负责自动化一轮核查；`quality_review` 的
`strict` 档位等同于人工复核。

### 9.3 禁止内容

AI 生成内容同样受《网络信息内容生态治理规定》第 7/8 条约束，禁止生成：
- 虚假信息
- 违反公序良俗内容
- 侵犯他人权益内容
- 违反国家安全相关内容

## 10. 典型违规案例库

近年来媒体内容违规的真实处罚案例（用于训练合规判断）：

### 10.1 时政类

**案例 A（2023）**：某商业媒体报道中央高层调研活动，擅自使用"据知情人士"
代替官方通稿，后经核实为**未授权释义**。处理：网信办约谈，责令平台下架，
涉事账号封禁 3 个月。

**借鉴**：时政稿件必须使用 S 级信源；不得使用"知情人士"等匿名来源作为
政治事实陈述依据。

### 10.2 广告法类

**案例 B（2024）**：某 MCN 机构在抖音种草视频中使用"全网最低价""绝对不
发胖"等极限词语。处理：市场监督管理局处罚 35 万元，全网账号封禁 7 天。

**借鉴**：`zhongcao_script` / `daily_zhongcao` 工作流必须调用
`compliance_check` 的广告法极限词扫描；命中立即阻断发布。

### 10.3 未成年人类

**案例 C（2024）**：某地方媒体报道未成年人校园霸凌事件，披露受害学生真
实姓名和就读学校。处理：网信办警告 + 责令更正 + 处罚 10 万元；记者停职检查。

**借鉴**：涉未成年人稿件必须经 `quality_review` 的未成年人保护扫描；命中
真实身份信息必须强制脱敏或拒发。

### 10.4 虚假新闻类

**案例 D（2024）**：某自媒体发布"某大厂 CEO 被带走调查"虚假消息，造成
股价大幅波动。处理：警方以编造、传播虚假信息罪立案；涉事账号永久封禁；
机构法人受刑事追诉。

**借鉴**：所有涉企业重大人事 / 股权 / 经营变动报道，必须经 `fact_check`
的 S/A 级信源确认；未确认一律按「传闻」或「待核实」表述。
