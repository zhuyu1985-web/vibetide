# Design: Refine Skill Management UX

## 参考分析：SkillsMP.com 设计模式

### 列表页关键设计
1. **行卡片布局**：每个技能占一行，水平排列所有信息
   - 左上：技能名称（醒目字体）+ 统计数字（stars 数）
   - 左中：来源/作者（头像 + 名称）
   - 左中：描述文本（完整显示，不截断）
   - 左下：更新日期 + 操作按钮
2. **排序切换**：Stars / Recent 两个排序方式
3. **搜索栏突出**：搜索框位于列表上方，宽度撑满

### 详情页关键设计（从页面结构推断）
- 独立路由页面，非弹窗
- 主内容区：完整渲染 Markdown 文档
- 侧边栏/顶部：元数据（来源、统计、日期等）

## 路由结构

```
/skills           → 列表页（改造现有页面）
/skills/[id]      → 详情页（新增）
  ├── page.tsx         — Server Component
  └── skill-detail-client.tsx — Client Component
```

## 数据需求

详情页需要新增的 DAL 查询：
- `getSkillDetail(id)`: 返回技能完整信息 + 绑定的员工列表（名称、ID、角色）
- 复用现有 `getSkillsWithBindCount()` 给列表页
