📊 数据模型 & 状态管理总览

Zustand Store (全局协调) 
interface ArticlePageStore {
  // 模式状态
  viewMode: 'read' | 'edit'
  contentType: 'article' | 'video'
  activeView: 'immersive' | 'web' | 'brief' | 'archive'

  // 面板状态
  leftPanelOpen: boolean
  rightPanelOpen: boolean
  leftTab: 'outline' | 'chat' | 'history'
  rightTab: 'analysis' | 'annotations' | 'transcript'
  zenMode: boolean

  // 跨组件联动
  selectedText: string | null
  selectedRange: { from: number; to: number } | null
  scrollToPosition: number | null
  highlightAnnotationId: string | null

  // 阅读外观
  appearance: AppearanceSettings
}
Feature Hooks (独立子系统) 
// 各 feature 独立管理自己的状态
useAIChat()        → 对话历史、流式状态
useAIAnalysis()    → 解读缓存、当前视角
useAnnotations()   → 批注列表、CRUD 操作
useArticleEditor() → Tiptap 实例、脏检查
useVideoPlayer()   → 播放状态、当前时间
useTranscript()    → 听记数据、同步高亮
useOutline()       → 大纲结构、滚动联动

// 跨 feature 联动通过 Zustand
// 例：选中文本 → store.setSelectedText
//    → AI Chat 读取 → 显示上下文
//    → Annotations 读取 → 创建高亮
