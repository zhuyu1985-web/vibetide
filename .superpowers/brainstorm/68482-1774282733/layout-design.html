新华社北京3月24日电　国务院日前印发《关于加快新能源产业高质量发展的指导意见》（以下简称《意见》），对新能源产业未来五年发展作出全面部署。

《意见》明确提出，到2030年，新能源消费比重达到25%以上，非化石能源发电装机占比超过50%。这标志着中国在"双碳"目标下，进一步加速能源结构转型步伐。

业内专家分析指出，此次政策的出台释放了三大信号：一是光伏、风电等可再生能源将获得更大规模的财政支持和税收优惠；二是储能技术研发被列为国家重点科技攻关方向...

🔑 布局核心规则

左侧栏 (20-25%)
  • Glass UI 面板背景 (backdrop-filter)
  • 可折叠，折叠后显示 AI 图标
  • 3 个 Tab：大纲 / 对话 / 历史
  • 底部固定输入框
中间核心区 (45-55%)
  • 纯净排版（无 Glass 效果）
  • 阅读区 max-width: 680px 居中
  • 编辑模式：Tiptap 全宽
  • 视频模式：自适应画幅播放器
右侧栏 (25-30%)
  • Glass UI 面板背景
  • 可折叠，折叠后显示批注图标
  • 动态 Tab：AI解读 / 批注 / 听记(视频)
  • 底部共享 AI 追问输入框
顶部导航栏 (h-12)
  • 面包屑 | 视图切换器 | 工具栏
  • 阅读进度条（顶部极细线）
  • 阅读/编辑模式切换按钮
  • 外观设置 / AI解读 / 批注 / 更多菜单
