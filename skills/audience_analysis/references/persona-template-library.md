## 2. 预定义画像模板

### 2.1 「Z 世代男性科技爱好者」

- 年龄：18-28
- 性别：男性为主
- 地域：一线 + 新一线
- 学历：本科及以上
- 职业：学生 / 程序员 / 互联网从业
- 活跃平台：B 站 / 知乎 / 小红书 / 少数派 / V2EX
- 阅读习惯：深度长文 + 视频评测
- 消费特征：愿为优质内容付费，对产品参数敏感
- 价值观：理性 / 实用主义 / 追求技术进步
- **典型内容**：科技产品评测、技术解析、创业故事

### 2.2 「一线白领都市女性」

- 年龄：25-40
- 性别：女性
- 地域：一线城市
- 学历：本科及以上
- 职业：金融 / 咨询 / 市场 / 设计
- 收入：中高
- 活跃平台：微信 / 微博 / 小红书 / 抖音
- 阅读习惯：通勤碎片化 + 周末深度
- 消费特征：品质生活 / 悦己消费 / 理财意识强
- 价值观：独立 / 平衡生活工作 / 重视仪式感
- **典型内容**：职场心得、健身美食、理财科普、旅行攻略

### 2.3 「三四线中产男性」

- 年龄：35-50
- 性别：男性
- 地域：三四线城市
- 职业：公务员 / 事业单位 / 企业中管 / 自主经营
- 收入：中等
- 活跃平台：微信 / 头条 / 快手 / 微博
- 阅读习惯：早晚通勤 + 夜间刷短视频
- 消费特征：家庭型消费 / 关注子女教育 / 重视品牌
- 价值观：稳定 / 家庭 / 社会责任
- **典型内容**：时政要闻、经济形势、养生健康、子女教育

### 2.4 「银发族长辈」

- 年龄：55+
- 活跃平台：微信 / 头条
- 阅读习惯：早起阅读 / 家人朋友圈转发
- 消费特征：保守 / 追求实惠 / 重视健康养生
- 内容偏好：健康科普 / 时政大事 / 家庭伦理
- 媒体素养：需要辟谣内容（易受谣言影响）

### 2.5 「小镇青年」

- 年龄：18-30
- 地域：四五线城市 / 县城
- 收入：中低
- 活跃平台：抖音 / 快手 / 小红书
- 消费特征：性价比导向 / 追赶潮流 / 消费能力渐增
- 内容偏好：短视频娱乐 / 影视综艺 / 生活技巧

### 2.6 「垂直意见领袖」（KOL / 行业专家）

- 活跃平台：微博 / 知乎 / B 站 / 公众号
- 特征：粉丝 1 万-100 万；垂类专家；内容有观点
- 消费特征：重视专业认可 / 对广告敏感
- 内容偏好：深度分析 / 行业报告 / 观点碰撞

## 3. 画像匹配算法

### 3.1 基于话题匹配受众

```python
def match_persona_by_topic(topic_keywords, personas):
    scores = {}
    for persona in personas:
        # 关键词与 persona 兴趣标签匹配度
        overlap = set(topic_keywords) & set(persona.interest_tags)
        score = len(overlap) / len(persona.interest_tags)
        scores[persona.id] = score
    return sorted(scores.items(), key=lambda x: -x[1])[:3]
```

def infer_audience_from_similar_articles(topic):

## 5. 时段画像差异

用户画像随时段变化：

### 早 7-9 点
- 上班族 + 学生
- 通勤场景
- 阅读习惯：短文 + 新闻简报
- 适合内容：时政要闻 / 行业动态

### 午 12-14 点
- 办公人群
- 休息场景
- 阅读习惯：短视频 + 中等长度
- 适合内容：娱乐 + 消费 / 种草

### 晚 18-22 点
- 全人群活跃
- 休闲场景
- 阅读习惯：深度长文 + 视频
- 适合内容：深度报道 / 评论 / 娱乐

### 夜 22-1 点
- 学生 + 夜班族 + 失眠者
- 私密场景
- 阅读习惯：情感内容 + 深度
- 适合内容：情感故事 / 心理 / 哲学

## 10. 注意事项

- **不得歧视**：任何画像都不应成为歧视依据（性别 / 地域 / 学历）
- **不得刻板印象**：画像是统计倾向，不是个体特征
- **隐私保护**：不得披露具体用户的个人画像
- **合规性**：涉未成年人画像需特别谨慎（《未成年人保护法》）
