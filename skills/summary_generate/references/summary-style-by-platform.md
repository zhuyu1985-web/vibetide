# 摘要生成 · 按平台差异化 Style Guide

> `summary_generate` 在不同分发渠道下的摘要风格差异。每个平台都有字数、语气、
> emoji 使用、标签规范的差异化要求。

---

## 1. CMS summary（正式入库摘要）

**字段**：`articles.summary` / CMS API `summary`
**字数**：80-150 字
**风格**：客观、信息量大、与导语互补不重复
**Emoji**：禁用
**结尾**：完整句号

### 示例
```
4 月 17 日，深圳市政府发布《关于促进人工智能产业高质量发展的若干措施》，
设立 200 亿专项基金，未来 5 年内规划建设 3 个国家级产业园，单企业最高可获
5000 万元补贴。此政策被业内视为深圳继基因工程后又一战略性产业布局。
```

---

## 2. 朋友圈分享摘要（微信）

**字段**：`shareSummary` / 微信开放平台 `desc`
**字数**：≤ 40 字（超长会自动截断）
**风格**：情绪化、有钩子、让人想点进来看
**Emoji**：可适度使用（1-2 个）
**结尾**：可省略句号（一句话感觉）

### 示例
```
🔥 深圳 AI 产业再出王炸政策 200 亿砸下来 谁能接得住？
```

---

## 3. 微博摘要（≤ 140 字）

**字段**：微博正文 body（不含链接和话题占位）
**字数**：100-130 字（预留 @ / # / 链接 10-20 字）
**风格**：新闻专业 + 话题感
**Emoji**：1-3 个（用于分段）
**结尾**：带 @机构 + #话题标签

### 示例
```
4 月 17 日 深圳 #AI 产业新政# 发布 📌 200 亿专项基金落地 📌 5 年建 3 个
国家级产业园 📌 单企最高补 5000 万 @深圳市人民政府 此轮力度前所未有
#人工智能#
```

## 4. 抖音 / 小红书摘要

**字段**：抖音视频 desc 或小红书笔记 body
**字数**：50-200 字（视版式而定）
**风格**：口语化、个性化、有 CTA
**Emoji**：3-5 个（视觉分段）
**结尾**：CTA + 话题标签

### 抖音示例
```
刚听说深圳又要搞 AI 大动作！💥
——— 200 亿专项基金到位
——— 5 年建 3 个国家级产业园
——— 单个企业最高补 5000 万
这力度你敢信？评论区聊聊 👇
#AI 产业 #深圳 #创业
```

### 小红书示例
```
📢 深圳 AI 新政来了 姐妹们冲！

✅ 200 亿基金 不差钱
✅ 5 年 3 个国家级产业园
✅ 单企最高 5000 万补贴

适合：AI 创业公司 / 技术人才想落户深圳的
不适合：传统行业转型还在观望的

#AI #深圳 #产业政策 #北漂深漂
```

## 5. TTS 播客摘要（listenSummary）

**字段**：播客音频前 15 秒口播
**字数**：60-80 字（按 180 字/分钟计 = 20-27 秒）
**风格**：对话体、简洁、有听感
**Emoji**：禁用（TTS 不读）
**结尾**：承接播客正文

### 示例
```
听众朋友们 今天聊一个重磅消息：4 月 17 日 深圳发布 AI 产业新政 设立
200 亿专项基金 单企最高补贴 5000 万 这是什么操作？我们一起来拆一下
```

## 6. RSS / 邮件订阅摘要

**字段**：RSS `<description>` / 邮件 subject / preview
**字数**：120-250 字
**风格**：信息密度极高、无花哨修饰
**Emoji**：禁用
**结尾**：句号 + 阅读全文 CTA

### 示例
```
深圳市政府于 2026 年 4 月 17 日发布《关于促进人工智能产业高质量发展的
若干措施》，核心内容包括：设立 200 亿元专项基金、5 年内规划建设 3 个
国家级人工智能产业园、单个企业最高补贴金额达 5000 万元。该政策被业内
视为深圳继基因工程之后又一次战略性产业布局。本文将深度解读政策亮点、
配套措施以及对深圳 AI 产业格局的深远影响。[点击查看全文]
```

## 7. SEO Meta Description

**字段**：HTML `<meta name="description">`
**字数**：140-160 字符（Google 截断阈值）
**风格**：含目标关键词、自然流畅
**Emoji**：禁用
**结尾**：不必完整句号

### 示例
```
深圳 AI 产业新政 2026 全文解读：200 亿专项基金落地 5 年建 3 个国家级
产业园 单企最高补贴 5000 万元 深度剖析政策亮点、配套措施及对产业格局
的影响
```

## 9. 字数精控方法

不同字数限制的裁剪策略：

### 从长到短（正文 → 摘要）

1. **第一轮**：提取核心句 + 关键信息（结构式摘要）
2. **第二轮**：保留 5W 要素 + 1 个数字
3. **第三轮**：押缩到 40 字内（朋友圈钩子版）

### 字数超限的 LLM 回退策略

```
if draft.length > max_length * 1.2:
    # 超限太多，重新生成
    regenerate(stronger_prompt_constraint)
elif draft.length > max_length:
    # 微超，硬截断最后一句
    trim_last_sentence()
else:
    # OK
    return draft
```

## 10. 质量自检清单（摘要专用）

- [ ] 字数符合目标平台要求
- [ ] 5W 要素覆盖（谁 / 做什么 / 何时）
- [ ] 核心数据保留
- [ ] 不剧透关键反转（短剧 / 综艺）
- [ ] 不命中广告法极限词（种草场景）
- [ ] 可独立阅读（不依赖正文上下文）
- [ ] Emoji 使用合规（按平台规范）
- [ ] 标签 / @ 符合平台要求
- [ ] 结尾完整（有 CTA 或句号）
