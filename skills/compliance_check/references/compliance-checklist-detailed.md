## 1. 广告法极限词（0 容忍 · 一票否决）

### 1.1 绝对化用语（《广告法》第 9 条）

| 词汇 | 典型搭配 | 替代 | 法规 |
|------|---------|------|------|
| 最 | 最好 / 最便宜 / 最大 | 非常 / 很 / 领先 | 9.3 |
| 第一 | 全网第一 / 销量第一 | 知名 / 热销 | 9.3 |
| 独家 | 独家首发 / 独家授权 | 重点推出 | 9.3 |
| 唯一 | 唯一选择 / 唯一品牌 | 主要 / 重要 | 9.3 |
| 顶级 | 顶级品牌 / 顶级服务 | 优质 / 高端 | 9.3 |
| 国家级 | 国家级推荐 / 国家级品质 | 优质 / 认证 | 9.1 |

### 1.2 权威误用

| 词汇 | 触发场景 | 处理 |
|------|---------|------|
| 央视推荐 | 未经央视正式授权 | 删除 |
| 国家xx品牌 | 未获得该称号 | 删除 |
| 中国驰名商标 | 商标法已取消 | 禁用 |

### 1.3 保证类极限

| 词汇 | 场景 | 替代 |
|------|------|------|
| 保证 | 保证有效 / 保证正品 | 致力于 / 努力 |
| 100% | 100% 有效 | 高度 / 显著 |
| 绝对 | 绝对安全 | 严格测试 |
| 无效退款 | 承诺范围不明 | 删除 |

## 2. 政治合规（0 容忍 · 一票否决）

### 2.1 领导人表述

- 姓名职务：严格按最新官方通稿；禁用简称
- 顺序：「职务 + 姓名」（新华体）如「总书记习近平」
- 首次出现 / 重要场合：必须完整「中共中央总书记、国家主席、中央军委主席习近平」

### 2.2 敏感话题禁区

| 禁区 | 典型表现 | 处理 |
|------|---------|------|
| 擅自使用领导人未公开讲话 | 「据知情人士透露，X 领导表示…」 | 立即删除 |
| 涉港澳台非规范表述 | 「台湾政府」 | 改「台湾当局」 |
| 民族宗教非中立 | 批评性表述 | 改中立 |
| 历史虚无主义 | 否定党的重大事件 | 删除 + 上报 |

### 2.3 国家重大节日 / 纪念日表述规范

- 国庆节 / 建党节 / 建军节 / 抗战纪念日 / 国家公祭日
- 禁用娱乐化语气：「开心 + 节日」「快乐 + 国难日」均违规
- 国家公祭日 (12.13) 24h 内所有内容应暂停娱乐性话题

## 3. 法律红线（硬违规）

### 3.1 《民法典》相关

| 条款 | 场景 | 处理 |
|------|------|------|
| 第 990 条 · 人格权 | 侮辱 / 诽谤 / 侵害肖像 | 必改或删除 |
| 第 1024 条 · 名誉权 | 未证实的负面描述 | 改为「据传 / 待核实」 |
| 第 1032 条 · 隐私权 | 披露个人隐私 | 脱敏 / 删除 |
| 第 1194 条 · 网络侵权 | 诽谤性转发 | 源链接断开 |

### 3.2 《刑法》相关

- 未经宣判不得称「罪犯」；可用「犯罪嫌疑人」
- 未判决案件不得断言「罪行」；用「涉嫌」
- 涉及国家安全 / 国家秘密 → 立即停止 + 上报

### 3.3 行业管理法规

| 行业 | 法规 | 合规点 |
|------|------|--------|
| 医疗 | 《医疗广告管理办法》 | 疗效保证 / 治愈率 禁用 |
| 药品 | 《药品管理法》 | 药品广告需审批号 |
| 金融 | 《证券投资基金法》 | 收益保证禁用 |
| 保险 | 《保险法》 | 收益确定性禁用 |
| 教育 | 《广告法》17 条 | 升学保证禁用 |

## 4. 伦理红线

### 4.1 未成年人保护

严格执行《未成年人保护法》第 49/51 条（详见 [media-industry-standards.md §6](../../../docs/skills/media-industry-standards.md)）。

### 4.2 反歧视

- 性别歧视：「女司机就是开不好车」
- 地域歧视：「XX 地方人素质低」
- 年龄歧视：「老人就是守旧」
- 职业歧视：「外卖员都是低学历」

全部标记 negative；建议改中性表达。

### 4.3 残疾人 / 疾病患者

- 禁用「残废 / 瘸子 / 瞎子 / 聋子」等歧视性称谓
- 改「残障人士 / 视障 / 听障」
- 精神疾病不得作为形容词用（「你神经病啊」）

### 4.4 反低俗

- 软色情 / 擦边
- 过度血腥 / 暴力描写
- 鬼神迷信宣扬
- 赌博 / 邪教
- 厌世 / 自杀暗示

## 6. 扫描策略

### 6.1 词库扫描（快速）

```python
def fast_scan(text: str, rules: dict) -> list:
    hits = []
    for rule in rules:
        for keyword in rule.keywords:
            if keyword in text:
                hits.append({
                    "keyword": keyword,
                    "rule": rule.name,
                    "severity": rule.severity,
                    "context": extract_context(text, keyword),
                })
    return hits
```

### 6.2 LLM 语义扫描（深度）

对于隐性违规（比如暗示 / 讽刺），词库扫不出来，需要 LLM：

```
system: 你是内容合规官，识别以下稿件是否存在隐性违规
user: [稿件内容]
output: { issues: [...] }
```

### 6.3 混合策略

- 第 1 轮：词库快速扫（100ms）
- 第 2 轮：疑似段落送 LLM 深度扫描（3-5s）
- 第 3 轮：高风险内容送人工复核（strict 档）

## 7. 档位策略

| 档位 | 扫描深度 | 拒发阈值 |
|------|---------|---------|
| **relaxed** | 仅词库扫 | 硬违规 ≥ 2 |
| **standard** | 词库 + LLM | 硬违规 ≥ 1 |
| **strict** | 词库 + LLM + 人工 | 任何命中硬违规 |

---

## 8. 违规严重度分级

| 级别 | 后果 | 处理 |
|------|------|------|
| 🚨 critical | 违法（国家安全 / 重大政治） | 立即拒发 + 上报 |
| 🔴 high | 硬违规（广告法极限词 / 政治红线） | 拒发 |
| 🟡 medium | 软违规（价值观偏差 / 措辞不当） | 人工复核 |
| 🟢 low | 风格问题 | 自动修正 / 建议 |
