# 内容生成 9 场景详细评分 Rubric

> 补充 SKILL.md §9 场景内容风格规范的详细评分标准。每项 0-10 分，
> 加总转换为 0-100 分。`quality_review` 的 `strict/relaxed` 档位据此打分。

---

## 通用 10 维评分（所有子模板适用）

| 维度 | 权重 | 10 分标准 | 5 分标准 | 0 分标准 |
|------|------|----------|---------|---------|
| 标题吸引力 | 10% | 含具体数字 + 关键事件 + 受众画像 | 平淡但准确 | 标题党或虚标 |
| 导语抓手 | 10% | ≤ 50 字、含 5W 核心要素 | 偏长但完整 | 超过规范字数上限 |
| 结构完整性 | 10% | 严格符合子模板结构要求 | 结构顺序有误 | 结构缺失或完全重构 |
| 事实密度 | 10% | 每 500 字至少 3 个可核查数字 / 人名 / 时间 | 事实偏少但准确 | 事实稀疏或错误 |
| 信源权威度 | 10% | 引用 S 级信源 ≥ 2 | 以 A-B 级为主 | 只有 C/D 级或无信源 |
| 专业术语准确性 | 10% | 100% 符合官方表述 | 1-2 处不规范 | 关键术语错误 |
| 语言流畅度 | 10% | 句式多变，无赘句 | 基本流畅 | 机器翻译腔 |
| 合规性 | 15% | 无任何违规命中 | 软违规 1 处（重复词） | 硬违规（命中极限词/政治红线） |
| 与 systemInstruction 匹配度 | 10% | 100% 执行指令要求 | 80% 匹配 | <50% 匹配 |
| 字数精控 | 5% | ± 5% 目标范围 | ± 15% 范围 | 超 ± 30% |

**总分计算**：加权和，阈值：
- `strict` 档位：≥ 95 才能发布
- `standard` 档位：≥ 85
- `relaxed` 档位：≥ 75

## 场景专属加分项 / 扣分项

### news_standard（新闻标准图文）

**加分**：
- +10：使用新华体「表示 / 指出 / 强调」系列动词
- +5：引述有完整机构职务 + 姓名 + 时间
- +5：关键数字有原始出处链接

**扣分**：
- -20：导语超过 100 字
- -30：使用情绪化动词（震惊 / 爆）
- -100：涉政治领导人姓名错误

### politics_shenzhen（深圳时政深度解读）

**加分**：
- +15：引用政策原文（加引号）≥ 3 处
- +10：会议 / 政策全称规范
- +10：对深圳具体政策背景有说明

**扣分**：
- -50：擅自评价政策（「力度不够」「有待观察」）
- -100：领导人称谓错误（职务 / 姓名 / 顺序）
- -100：使用「开会」等非正式动词
- -100：敏感话题私自延伸

### sports_chuanchao（川超赛事战报）

**加分**：
- +10：比分 + 进球时间精确到分钟
- +10：关键球员数据（射门 / 传球成功率 / 控球时间）
- +5：战术点评有具体阵型（4-3-3 / 3-5-2）

**扣分**：
- -50：比分错误
- -50：球员姓名错误
- -100：地域攻击 / 球员人身攻击 / 裁判黑哨定性

### variety_highlight（综艺盘点图文）

**加分**：
- +10：Top N 盘点逻辑清晰（按出场顺序 / 热度 / 话题度）
- +5：艺人名准确（KB 核查通过）

**扣分**：
- -50：艺人名错误（KB 核查未过）
- -100：捏造艺人黑料 / 绯闻

### livelihood_zhongcao（种草图文）

**加分**：
- +10：钩子 + 痛点 + CTA 结构完整
- +10：合作披露明显（开头「本文由 XX 赞助」）
- +5：使用 emoji / 网感词汇适合平台

**扣分**：
- -50：命中广告法极限词（最 / 第一 / 绝对）
- -50：未披露合作关系但明显推广
- -100：推销医疗器械 / 药品（不允许）

### livelihood_tandian（探店图文）

**加分**：
- +10：地址精确到街道 + 门牌 + 地图链接
- +10：人均消费精确到元 + 标注时间
- +5：营业时间 / 是否需要预约 / 停车信息

**扣分**：
- -30：地址模糊或错误
- -50：价格虚假（高于 / 低于实际 ≥ 20%）
- -100：虚构试吃体验

### livelihood_podcast（播客文字稿）

**加分**：
- +10：可读性强（长句改短句）
- +10：与音频时长匹配（180 字/分钟语速）
- +5：话题标记（#每日AI播客）

**扣分**：
- -30：文字稿有冗余但音频没讲
- -100：政治敏感话题处理不严（严档要求）

### drama_serial（短剧文字稿）

**加分**：
- +10：CMS 适配结构（章节 / 集 / 简介）
- +5：标签丰富（类型 / 主演 / 题材）

**扣分**：
- -20：剧透关键反转
- -100：涉未成年人不当情节

### daily_brief（每日简报图文）

**加分**：
- +10：3-5 条热点覆盖多领域（政治 / 经济 / 科技 / 文体）
- +5：每条有「一句话看点」
- +5：开场引导语有信息量

**扣分**：
- -30：条目同领域霸榜（如 5 条都是 AI）
- -50：有一条是过期新闻（≥ 24 小时）

## 打分示例

**稿件**：`news_standard` 子模板，1500 字，深圳 AI 产业新政报道

| 维度 | 分数 | 说明 |
|------|------|------|
| 标题吸引力 | 8/10 | 「深圳发布 AI 产业新政策 设立 200 亿专项基金」- 有数字但缺受众画像 |
| 导语抓手 | 9/10 | 85 字，含 5W |
| 结构完整性 | 9/10 | 倒金字塔 + 5W1H 覆盖 |
| 事实密度 | 9/10 | 每 500 字有 4 个具体数字 |
| 信源权威度 | 10/10 | 新华社 + 深圳市政府官网 |
| 专业术语 | 10/10 | 官方表述准确 |
| 语言流畅度 | 8/10 | 基本流畅，2 处句式单一 |
| 合规性 | 15/15 | 无违规 |
| systemInstruction 匹配 | 10/10 | 100% 执行 |
| 字数精控 | 5/5 | 1502 字在 1500 ± 5% 内 |

**总分：93（standard 档通过，strict 档需优化标题）**

修改建议：
- 标题加入受众词：「深圳 AI 新政：初创团队最高获 5000 万补贴」
- 「基本流畅」段落改用长短句搭配
