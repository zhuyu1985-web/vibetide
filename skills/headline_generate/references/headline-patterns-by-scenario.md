# 标题生成 · 场景 × 句式模式库

> 每个场景的「好标题」句式模式 + 反面案例，用于 `headline_generate` 的
> variant A/B/C 生成时参考。每条标注字数范围、适用词频、心理锚点。

## 1. news_standard（新闻标准）

### 推荐句式

| 句式 | 示例 | 心理锚点 | 字数 |
|------|------|---------|------|
| 主语 + 动词 + 宾语 + 数据 | 深圳发布 AI 产业新政 设立 200 亿专项基金 | 权威 + 具体 | 16-24 |
| 【时间】主语 + 动作 + 结果 | 4 月 20 日 国务院发布 AI 产业扶持意见 | 时效 | 18-22 |
| 数字开头 + 核心事件 | 200 亿专项基金落地 深圳 AI 产业加速成型 | 惊奇 + 权威 | 16-24 |
| 关键动词 + 全名机构 + 表态 | 人民银行决定：自 5 月 1 日起下调 LPR | 官方可信 | 18-26 |

### 反面案例

| 错误标题 | 问题 | 改正 |
|---------|------|------|
| 重磅！深圳搞了个大新闻 | 标题党 + 信息空洞 | 深圳发布 AI 产业新政 设立 200 亿专项基金 |
| 深圳最屌的政策来了 | 极限词 + 不规范 | 深圳出台力度空前的 AI 产业扶持政策 |
| 据传深圳要大搞 AI | 信源模糊 | 深圳市政府发布 AI 产业发展规划（附原文） |

## 2. politics_shenzhen（深圳时政）

### 推荐句式

| 句式 | 示例 | 约束 |
|------|------|------|
| 【会议全称】决定/审议/通过 + 内容 | 深圳市委八届七次全会通过《……》 | 机构名全称 |
| 领导人 + 动词（考察/调研/会见） + 地点 + 表态 | 市委书记深入前海调研 强调高水平开放 | 姓名+职务准确 |
| 【重要政策】发布 + 全称 + 核心 | 《促进 AI 产业高质量发展若干措施》今日发布 | 引号包原文 |

### 必守红线

- 标题中**绝不简写**机构或政策全称（首次出现）
- **领导人姓名必须置于职务之前**（新华体）
- 避免任何评价性词汇（「力度不够」「有待观察」）

### 反面案例

| 错误 | 问题 | 改正 |
|------|------|------|
| 深圳市委开大会 定调 AI 发展 | 动词不规范（开大会） | 深圳市委召开全会 部署 AI 产业发展 |
| 张三书记：要大力发展 AI | 职务在姓名后 | 市委书记张三：扎实推进 AI 产业高质量发展 |
| 这个政策太牛了 | 评价性 | 《……》发布 政策亮点凸显 |

## 3. sports_chuanchao（川超赛事）

### 推荐句式

| 句式 | 示例 | 情绪强度 |
|------|------|---------|
| 比分 + 赛事 + 赛果 | 3-1！川超揭幕战成都蓉城力克四川九牛 | 中 |
| 关键球员 + 动作 + 结果 | 张三梅开二度 成都蓉城主场胜川队 | 高 |
| 悬念/反转 + 结局 | 落后 2 球逆转 成都蓉城绝杀保级 | 爆点 |
| 数据 + 评论 | 控球 70% 射门 18-4 成都蓉城大胜四川九牛 | 中 |

### 词汇偏好

**推荐**：力克 / 险胜 / 大胜 / 完败 / 逆转 / 绝杀 / 补时 / 绝平 / 扳回 / 压哨

**禁用**：暴虐 / 血洗 / 碾压（负面暴力词）

### 反面案例

| 错误 | 改正 |
|------|------|
| 成都蓉城血洗四川九牛 | 成都蓉城大胜四川九牛 3-1 |
| 四川九牛是垃圾 | 四川九牛开局不利 需调整阵容 |
| 裁判黑哨放水 | 部分判罚引发现场球迷讨论 |

## 4. variety_highlight（综艺盘点）

### 推荐句式

| 句式 | 示例 | 适用 |
|------|------|------|
| Top N + 盘点 + 核心 | 《XX》第 5 期 5 个名场面盘点 | 热播综艺 |
| 艺人名 + 动作 + 反应 | 赵大爷暖心行为 引全场起立鼓掌 | 情感性 |
| 数字 + 惊人点 + 节目名 | 收视率破 3！《XX》连续 3 期登顶 | 数据向 |

### 必守

- 艺人名必须 KB 核查通过（姓 + 名完整）
- 不得使用艺人昵称（除非节目内主推的昵称）
- 不得捏造艺人绯闻 / 丑闻作为标题

### 反面案例

| 错误 | 问题 | 改正 |
|------|------|------|
| 某 L 姓男星节目中突发情绪失控 | 化名暗示 + 推测 | 《XX》录制现场 李某情绪自然流露引关注 |
| 塌房了！这个明星人设崩了 | 网络俚语 + 诽谤 | 某艺人舞台表现引发网友讨论 |

## 5. livelihood_zhongcao（种草）

### 推荐句式

| 句式 | 示例 | 平台适配 |
|------|------|---------|
| 个人体验 + 产品 + 效果（非极限） | 油皮姐妹亲测！这款粉底让我卡粉痕迹明显改善 | 小红书 |
| 钩子 + CTA | 换季护肤 3 步走 这个组合我回购了 3 次 | 抖音 |
| 数字 + 痛点解决 | 30 秒搞定午餐便当 这 3 款食材救了我 | B 站 |

### 广告法合规

- **禁用**：最 / 第一 / 绝对 / 顶级 / 国家级 / 100%（广告法极限词）
- **必加**：「（个人使用感受）」「（仅代表个人体验）」等声明
- **合作披露**：开头「本文由 XX 赞助」+ 标签 #合作

### 反面案例

| 错误 | 违反 | 改正 |
|------|------|------|
| 全网最便宜的粉底！绝对首选 | 广告法极限词 | 这款粉底我回购了 3 次 性价比真的高 |
| 100% 保湿 不卡粉 | 绝对化保证 | 油皮亲测保湿持久 上妆贴合不卡粉 |

## 6. livelihood_tandian（探店）

### 推荐句式

| 句式 | 示例 |
|------|------|
| 城市 + 店名 + 亮点 | 成都春熙路 | 这家串串火到排队 3 小时 |
| 人均 + 味道评价 | 人均 ¥80 吃到撑 成都宽窄巷子新晋烤鱼店 |
| 探店结论 + 推荐度 | 踩雷预警 × 回本机率 90%！成都这家日料我回 3 次 |

### 合规要点

- 地址精确到街道 + 门牌号
- 人均消费精确到元，注明「（2026-04 参考价）」
- 合作类明确标 #合作

## 多版本差异策略（A/B/C）

`headline_generate` 默认输出 3 个 variant：

- **variant_a（专业版）**：新华体严肃准确，无情绪词，数据驱动
- **variant_b（网感版）**：适合社交平台分享，情绪词适度，有反差 / 悬念 / 数字
- **variant_c（悬念版）**：疑问句 / 悖论 / 欲知后事，适合阅读引导

**三者字数差异**：variant_a 最简短，variant_b 中等，variant_c 可稍长（但仍在
scenario 约束内）。

## 2026 年常见标题违规红线速查

| 问题 | 触发频率 | 处理建议 |
|------|---------|---------|
| 极限词（最 / 第一 / 顶级） | 种草场景高频 | 改为「我的最爱」「让人印象深刻」 |
| 政治敏感（领导人姓名错误） | 时政场景 | 对照最新官方通稿 |
| 谣言 / 未证实消息 | 社会新闻 | 加「疑似」「传闻」「待核实」 |
| 诋毁 / 攻击 | 体育 / 娱乐 | 用中性词 |
| 标题党（内容不符） | 种草 / 娱乐 | 回到正文找真实卖点 |
