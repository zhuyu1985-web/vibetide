## 2. 最佳发布时机

### 2.1 早间档（6-9 点）

- 目标：通勤族碎片阅读
- 推荐平台：微信头条 / 头条号 / 抖音
- 内容形态：短资讯 / 早报
- **黄金时段**：07:30 / 08:30

### 2.2 午间档（11-14 点）

- 目标：午休放松
- 推荐平台：小红书 / 微博 / 抖音
- 内容形态：轻松图文 / 短视频
- **黄金时段**：12:30 / 13:00

### 2.3 下班档（17-19 点）

- 目标：离线阅读
- 推荐平台：微博 / 微信 / 头条
- 内容形态：深度长文 / 分析
- **黄金时段**：18:30 / 19:00

### 2.4 晚间档（20-23 点）

- 目标：全人群活跃期（流量峰值）
- 推荐平台：抖音 / B 站 / 小红书 / 视频号
- 内容形态：视频为主
- **黄金时段**：20:30 / 21:30

### 2.5 深夜档（23-1 点）

- 目标：失眠族 / 夜班族
- 推荐平台：微博 / 抖音 / B 站
- 内容形态：情感 / 深度 / 音频
- **黄金时段**：23:30

## 3. 账号矩阵策略

### 3.1 头部账号

主品牌旗舰号，覆盖所有核心内容：
- 微信头条
- 微博主账号
- 抖音主账号
- 今日头条主账号

### 3.2 垂直账号

细分领域账号：
- 科技号（AI / 数码 / 互联网）
- 民生号（本地 / 社会）
- 娱乐号（影视 / 综艺 / 明星）
- 体育号（赛事 / 球队 / 运动员）
- 财经号（股市 / 理财）

### 3.3 矩阵化发布

同一稿件，多账号差异化发布：
- 主账号：原稿 / 完整版
- 垂直号：聚焦版（只保留自己垂类相关）
- 短视频号：视觉化（脚本改写）

## 4. 平台内容适配

### 4.1 微博（≤ 140 字 / 可长微博）

- 一般用短微博 + 话题标签 + 图片
- 长微博要点：第一句抓人 + 段落清晰
- 示例：
  ```
  🔥 深圳 AI 新政实锤了！
  
  📌 200 亿专项基金
  📌 5 年建 3 个国家级产业园
  📌 单企最高补贴 5000 万
  
  @深圳市人民政府 #AI产业#
  ```

### 4.2 微信公众号（长文 2000+ 字）

- 标题：吸引点击（≤ 20 字）
- 摘要：8-10 字精华
- 正文：结构化 + 图文并茂 + 小标题
- 文末：引导关注 + 互动

### 4.3 抖音（60s 短视频 + 文案）

- 前 3 秒：抓人钩子
- 中间 45 秒：核心内容
- 结尾 12 秒：CTA + 关注提示
- 文案：口语化 + Emoji + 标签

### 4.4 小红书（400-600 字图文）

- 封面：有梗的大图
- 标题：利用关键词 + 好奇心
- 正文：表情分段 + 重点加粗
- 结尾：话题 + 关注

### 4.5 B 站（5-8 分钟视频 + 稿件）

- 封面：高质量 + 悬念
- 稿件结构：开场 30s + 正文 3 章 + 结尾 CTA
- BGM 选择重要（影响完播率）

## 5. 效果预估

基于历史数据预估（参考区间，非承诺）：

### 5.1 微博

- 阅读：粉丝数 × 10% ~ 50%（看转发率）
- 热门稿件：万-百万
- 超热稿件：千万级

### 5.2 微信公众号

- 阅读：关注数 × 20% ~ 40%
- 头条稿件：优质 1 万-10 万
- 爆款：10 万+

### 5.3 抖音

- 播放：根据账号热度 5 千-100 万
- 爆款：1000 万+
- 受算法波动影响极大

## 7. 发布后监控

### 7.1 立即监控（0-1h）

- 查看初始打开率
- 评论情感（positive / negative 比例）
- 是否引发舆情

### 7.2 短期监控（1-24h）

- 阅读 / 互动达到预期区间
- 转发路径（大 V / KOL 参与）
- 竞品对标

### 7.3 中期监控（1-7d）

- 长尾效应
- 二次传播
- 对品牌形象影响

## 8. 紧急预案

### 8.1 稿件舆情发酵（负面）

- **1 小时内**：@公关 + 总编
- **2 小时内**：决定是否撤稿 / 修改
- **4 小时内**：官方声明（如需）

### 8.2 数据错误被发现

- **30 分钟内**：更正 + 置顶致歉
- **2 小时内**：跨平台同步更正
- **次日**：发布更正说明稿

### 8.3 合规被投诉

- **立即**：下架 + 内部调查
- **2 小时内**：回应投诉方
- **24 小时内**：整改方案
